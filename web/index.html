HomeOps Guides using Devops Tools

Welcome to the inetfish.com website.
This website is a collection of guides and tools for implementing automation and tools in the home environment.
The home network and home computing environment has become more complex over the years and this creates a few opportunities as well as problems to be solved. I call it homeops.

This homeops project is divided into 3 main categories:
  1. Infrastructure - The supporting infrastructure to run various aspects of the homeops project
  2. HompeOps - A Collection of tools to automate the operational activities in the home network and computing environment.
  3. Web Tools - Additional tools that I found useful to develop and can be hosted in the home ops enviornment, or used directly from this website.
Guiding principles
  1. Automation -- Homeops project and scripts should be fully automated.
  2. Keep it Simple -- Homeops follows standard software design and operational practices. Web design is geared towards performance and simplicity.
  3. Low Cost/No Cost -- Aside from a desktop computer, Windows OS, and Internet connection -- there are no additional costs.
  4. No Advertising -- Advertisiting is forbidden.
