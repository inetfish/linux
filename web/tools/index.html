HopeOps Tools: What's My IP, MOTD, more..

*Calling the tools programatially will return the simple result without ipbob text.
Tools List
  1. What's my IP -- Returns your public IP; usage: http://ipbob.com/myip
  2. motd -- Returns a daily inspirations, changes daily; usage: http://ipbob.com/motd
  3. rnd -- Returns a random number; usage: http://ipbob.com/rnd/min/max/ :default: 1/100
  4. curl -- Returns http status code for a URL; usage: http://ipbob.com/curl/website :default: yahoo.com
  5. Ping -- Returns ping result; usage: http://ipbob.com/ping/address :default: fail
  6. strlen -- Returns length of given string; usage: http://ipbob.com/strlen/string :default: invalid
  7. strlo -- Returns lower case of given string; usage: http://ipbob.com/strlo/string :default: invalid
  8. strlen -- Returns upper case of given string; usage: http://ipbob.com/strup/string :default: invalid
