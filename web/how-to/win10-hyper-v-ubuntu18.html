In my enviornment, my host computer is windows 10 running Hyper-V.

  1. Ubuntu 18.04 VHD Download
  2. Create a folder for your new virutal machine
  3. Make a copy of your VHD from step 1
  4. Create a new VM using Hyper-V manager
    • new | virtual machine
  5. Specify the name and location
  6. Select G2 as the generation
  7. Assign memory: 1024/Dynamic
  8. Connect Virutal Hard Disk
    • Select use an existing & the vhd from #3
      • Finish
The new VM will now be in the hyper-v manager.
  1. Right click the VM, select settings
  2. Disable secure boot
Now the VM can be started.
