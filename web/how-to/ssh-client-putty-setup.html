SSH client setup for windows using putty and putty connection manager

Install Putty or just download putty.exe from the putty website.
  1. Download Putty
After downloading is complete, configure Putty
  1. start putty client, from windows start menu, type putty.
    • create a desktop shortcut to putty.exe if it is your preference.
  2. Enter the host ip/name. Enter in the format user@ip
  3. Enter a name for the session and save the session.
  4. Open the session to verify the ssh session is connecting.
Install Putty Connection Manager. Instructions found here:
  1. Download Putty Connection Manager
  2. Download to a known folder in the system. Downloads folder is ok.
  3. Extract the zip.
    • In the reference system, we have extracted the puttycm.exe to the putty folder for simpler reference in the future.
  4. For launching puttycm, browse to the install location and right click, create a shortcut, put the shortcut on the desktop.
  5. puttycm can also be launched by typing puttycm into the start menu.
