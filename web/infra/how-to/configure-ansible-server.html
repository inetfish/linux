Ansible Server Setup on Ubuntu 18.04

System Setup
  1. Login as the ansible user
  2. Change the system hostname
    • sudo hostnamectl --no-ask-password set-hostname
    • The reference system hostname is ansible1.
    • reboot.
Configure Ansible User Private Key
  1. Copy ansible private key to /ansible1/.ssh/id_rsa
  2. Set permissions on private key to 600.
  3. Verify key setup by ssh to docker1.
Configure GIT
  1. Create a src directory in the ansible user home directory.
    • cd ~ ; mkdir src
  2. Install git and clone the inetfish source code.
    • sudo apt-get update; sudo apt-get install git
    • git config --global user.name
    • git config --global user.email
    • git clone https://github.com/opsgriller/inetfish.git
    • Note:Must have a github account.
    • chmod all scripts to chmod 750
Install Ansible server using inetfish scripts
  1. ./install-universe
  2. ./install-ansible
  3. ./copy-ansible-hosts
