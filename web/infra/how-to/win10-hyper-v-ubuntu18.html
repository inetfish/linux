Setup Ubuntu virtual machine(s) using Windows10 Pro & Hyper-v.

The reference environment is Windows 10 Pro host running Hyper-V as the base foundation for the VMs.
Follow the instructions to setup VMs as needed.
  1. Ubuntu 18.04 VHD download -- See VHD download step for VHD details
  2. Create a new folder for your new virutal machine
  3. Make a copy of the VHD ,from step 1, into the new folder
  4. Start Windows 10 Hyper-V manager to create a new virtual machine
    • Start menu | Type Hyper-V and select it from the start menu.
  5. From the Hyper-V app, Right click on your server | new | virtual machine
  6. Specify the name and location
  7. Select G2 as the generation
  8. Assign memory: 1024/Dynamic
  9. Select WAN for network
  10. Connect Virutal Hard Disk
    • Select 'Use an existing virtual hard disk' & select the vhd location from step #3
    • Next to verify the settings.
    • Finish
The new VM will now be in the hyper-v manager. For it to boot, disable secure boot option.
  1. Right click the VM, select settings
  2. Security | Uncheck - Enable Secure Boot
Now the VM can be started.
